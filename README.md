# hexo-theme-zen
## 禅 - Hexo极简主题

---

功能简单，但是文件超小，不开评论功能每个页面只加载**15kb**左右

配置文件参考`_config.zen.yml`

支持`Twikoo`评论功能

正确的高亮配置
```yaml
highlight:
  enable: false
  line_number: true
  auto_detect: false
  tab_replace: ''
  wrap: false
  hljs: false
prismjs:
  enable: true
  preprocess: true
  line_number: true
  tab_replace: ''
```
